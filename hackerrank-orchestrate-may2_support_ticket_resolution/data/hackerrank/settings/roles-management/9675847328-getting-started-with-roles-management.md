# Getting Started with Roles Management

_Last updated: Mar 5, 2025, 11:10 AM (Last updated 1 year ago)_

## Roles on HackerRank

A **Role** represents a job position at an organization that is a collection of skills aligned with industry standards. HackerRank, guided by its **Skills Directory**, has curated high-quality questions mapped to these skills to assess candidates accurately.

### Roles Management

The HackerRank for Work platform's Roles Management module provides a structured and customizable process for managing roles and streamlining the overall hiring workflow. 

### Accessing Roles Management

On the HackerRank platform, click the profile icon and select **Roles Management **from the dropdown.

### Custom Roles

Custom roles allow organizations to tailor roles according to specific hiring needs. Only company admins can create roles within their accounts. Once created, these roles are available for all users in the organization.

**Steps to Create a Custom Role:**

- Click **Create Role**.

- In the popup, provide a role name associate relevant skills, and save or

- Use **Save to Draft button** to save the role without publishing it.

### Managing Roles

#### **Accessing and Using Roles**

- View all available roles (default and custom) when creating a test or scheduling an interview.

- Select a role from the list or search using the **search bar** at the top.
