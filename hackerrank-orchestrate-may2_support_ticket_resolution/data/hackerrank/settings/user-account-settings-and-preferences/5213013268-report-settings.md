# Report Settings

_Last updated: Dec 26, 2024, 3:19 PM (Last updated 1 year ago)_

You can set up or change your preferences for the Microsoft Excel CSV reports generated for the candidates. These reports can be downloaded from the Test Reports page. The preferences you set here are reflected in the Microsoft Excel CSV reports you downloaded from the HackerRank platform. 

## Accessing Report Settings

- On the top right corner of your HackerRank account home page, you can access Settings. 

- You can find the Report Settings placed on the left pane under User settings on the settings page.

## Components of Report Settings

Report settings are divided into six sections, each with fields you can select or unselect by ticking the check box attached to it. 

### Candidate

In this section, you can select the candidate fields you want to include in your report sheets, such as their Email, ID, Name, Tag, and IP Address.  

### Company

This section allows you to include your Company ID and Company Name in your reports.  

### Test 

There are several fields in the Test section. The definitions of each field are given in the table below to help you better understand when choosing an option.  

|  |  |
|----|----|
| **Term ** | **Definition** |
| Invite ID | Invitation ID is generated while inviting a Candidate.  |
| Invite Method | Method of Invitation  |
| Invite Valid From | The date and time from which the Invitation allowed the candidate to take part in a test. |
| Invite Valid To | The date and time when the Invitation gets invalid.  |
| Invited At | Time of the Invitation |
| Invited By Email | Email of the recruiter who invited the Candidate.  |
| Invited By Name | Name of the recruiter who invited the Candidate.  |
| Role | the test is created for.   |
| Test Duration | The duration set for the Test.  |
| Test ID | The ID of the particular Test.  |
| Test Name | Name given to the Test |
| Test Owner Email | Email of the Test creator.  |
| Test Owner Name | Creator of the Test. |
| Test Tags | Tags given for the Test.  |

### Attempt Settings 

The definition of each field related to a candidate’s attempt has been given in the table below.

|  |  |
|----|----|
| **Term** | **Definition** |
| Attempt Count | The number of attempts made by the candidate.   |
| Attempt Endtime | Ending time of the candidate’s attempt.   |
| Attempt ID | The ID of the candidate’s attempt |
| Attempt Plagiarism | Plagiarism in Attempt |
| Attempt Start time | Start time of the Candidate’s attempt. |
| Attempt Time Taken | Time taken to finish a test.  |
| Cutoff Score | The cutoff score of the test taken by the candidate.  |
| Detailed Report Url | Url to access the detailed report of the candidate’s attempt.  |
| Feedback Comments | Feedback comments given by the candidate at the end of the test.  |
| Last Attempt Date And Time | Candidate’s last attempt date and time.  |
| Max Score | The maximum score attained by the candidate  |
| Network Disconnected Duration | Duration of the time when candidate lost their network |
| Number Of Window Exits | The number of times the candidate moved away from the test window. |
| Out Of Window Duration | Duration the candidate spent away from the test window.  |
| Percentage Score | Percentage of the score attained by the candidate.  |
| Proctor Flag | Flags raised by the HackerRank proctoring system. |
| Proctor Flag Reasons | Reasons behind a candidate getting the proctoring flags.   |
| Product Rating | Rating given by the candidate at the end of the test.  |
| Score | The final score attained by the candidate.  |
| Status | Status of the candidate’s attempt |
