# Invite a New User Prerequisite Inviting a new user

_Last updated: Feb 13, 2026, 3:36 PM (Last updated 2 months ago)_

You can invite new users to HackerRank for Work account and assign roles and team access.

**Note**: To add an existing user to a team, see [📄 Inviting an Existing User to a Team](/articles/2203617737).

# Prerequisite

You must have Company Admin access.

# Inviting a new user

To invite a new user:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Select your profile icon in the upper-right corner.

3.  Select **Teams Management** from the drop-down menu.

4.  Click **Add New User** under the **Users** tab.

5.  In the **Add New User** dialog, select one of the following options:

    - **Email Invite:** Invite a single user by email.

      ![email2.png](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1770975144634-email2.png?Expires=253370764800&Signature=JbrbVQEAYrBb7U7O5jPJYl5VaWc6coNkR6VrT4~gnxz6p~xZrMGIh6ZRpZp-K6pGmg3Mpf1yk5ojX2Q6RH8f565cTwxFaSZ2WST15FKEHHA777J8BQ4gVpl4F2jbWvRRhirLsBBya3geueMGBC-0CcHrOslJ2D13tT56w~C5-6tBpF-rSgM6Xo3hPUoUFVWuirGK5cuSk3FJH0CrPYYKN3F3NCyrhxRRQNr2oHL~k7qlP9tszSWFirOnPH0B9379UUkQ91RgDrF0BmLR7vjOGLupKKVp-2rpDBTKRBuABXxMrZZnIIYdkmtcBvbyOeK6wlFuxNfS6SNAm9kZ46evGg__&Key-Pair-Id=K3NV4LZ47N8M46)

      1.  Select a user role from the **User Role** drop-down list. For more information about user roles, see [📄 Types of User Roles](/articles/9603546665).

      2.  Enter the user’s name and email address.

      3.  Select one or more teams from the **Teams** drop-down list.

      4.  Click **Add**.

      HackerRank sends an invitation email to the user.

    - **Link Invite:** Generate an invitation link and manually share it.

      ![link2.png](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1770975166793-link2.png?Expires=253370764800&Signature=ebr7ni5qdd~ZIShXfMoxaMGw~IRUwpOD3DMGnt1JKBMhcEBSCu7OUwGVcic7OMWh3Ba8C1lS3oipTtTG4tH-6w~mhhHsmuKXasRBY92GXjr96nfdIcRHTr-LOHJIUldvmsU4CjPM69szXO~ol-dWBnDwqIzCaHGEqy9SB~cl77KKbtCGRTedOcMwEoI7cSfKYzw3FkLoyNXatNKM0azvi2G0dqNJzRyE53c0C62JRptsOmUT36ikh0EdRd4w4FO7eQS4ydtfJnN1CKYAgyFP8IhpWn20XPOuVPPxj0J0VKCL4xVvHf6r73lTyvB6STvK~RKWBSbATjKQC~via5CfAw__&Key-Pair-Id=K3NV4LZ47N8M46)

      1.  Select a user role from the **User Role** drop-down list. For more information, see [📄 Types of User Roles](/articles/9603546665).

      2.  Select one or more teams from the **Teams** drop-down list.

      3.  Click **Copy Link** and share the link with the user.

    - **Bulk Invite:** Invite multiple users at once using a CSV file.

      ![bulk.png](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1770975180983-bulk.png?Expires=253370764800&Signature=Oq00wZsbLHrC8SVguuypxO1YknkrHb1vCqwz7Q9fQJDYyv9uARE29T7Hqp7K9DfMhKaQJDc4~qWnXysKhT7enBRafcGXSQF6FQuJ~kvh07~m0AdRVOpXdNN86deyDrJiCi7aOcGafJFsLMP3BeEo9MWuaOJphuHlGyy8m~P~HHSSf8x3AEAs0ypZnQnGRww5SgKf4dlU0ouLrrPx~DDCvHA7RD-f1tSTmng0Y-gAC7L7~57SCsFmN7cMOxz9NA50-vKxezszSy3bekLPIBscaC6tgIAbfZ0jsQiNWogKVzj472YbvK2yemOagyfugwHEWweCZqf1vGUaCZ~XEu0-Zw__&Key-Pair-Id=K3NV4LZ47N8M46)

      1.  Click **Download Sample CSV file** to view the required format and create your CSV file.

      2.  Click **Select a file** to upload the CSV file, or drag and drop the file.

      3.  Click **Add**.

      HackerRank sends invitation emails to all users listed in the CSV file.
