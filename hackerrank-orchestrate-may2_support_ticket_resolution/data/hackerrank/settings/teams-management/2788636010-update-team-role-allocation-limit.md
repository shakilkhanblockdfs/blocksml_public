# Update Team Role Allocation Limit Prerequisite Updating a team role allocation limit

_Last updated: Mar 3, 2026, 3:35 PM (Last updated 2 months ago)_

HackerRank allows you to configure role allocation limits for an existing team to control how many users you assign to each role.

# Prerequisite

You must have Company Admin or Team Admin access.

# Updating a team role allocation limit

To update a team role allocation limit:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Select your profile icon in the upper-right corner.

3.  Select **Teams Management** from the drop-down menu.

4.  Go to the **Teams** tab.

7.  Select **No limit** or choose a numeric limit value from the dropdown for each of the following roles:

    - **Recruiter Limit**

    - **Developer Limit**

    - **Company Admin Limit**

    - **Interviewer Limit**

For more information about roles, see [📄 Types of User Roles](/articles/9603546665).

The system automatically saves all changes.
