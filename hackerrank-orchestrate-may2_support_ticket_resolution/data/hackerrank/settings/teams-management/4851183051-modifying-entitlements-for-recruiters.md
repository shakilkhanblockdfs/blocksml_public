# Modify Entitlements for Recruiters Prerequisite Modify entitlements for recruiters Available recruiter entitlements

_Last updated: Jan 28, 2026, 11:00 AM (Last updated 3 months ago)_

You can modify recruiter entitlements to control access to tests, interviews, questions, and candidates.

# Prerequisite

You must have created a team.

# Modify entitlements for recruiters

To modify entitlements for recruiters in a team:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Select your profile icon in the upper-right corner.

3.  Select **Teams Management** from the drop-down menu.

# Available recruiter entitlements

You can manage the following entitlement categories for recruiters:

- Tests

- Interviews

- Questions

- Interview templates

- Interview scorecards

- Candidates

- Exports

## Tests entitlements

  - Send Test Invites

  - Edit Tests

    - Clone Tests

    - Delete Tests

- View Test Insights

- View Test Settings

  - View Test Invites Settings

  - View Test General Settings
