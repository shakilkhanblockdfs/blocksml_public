# Lock User Access Prerequisite Locking user access Unlocking user access

_Last updated: Apr 23, 2026, 1:21 PM (Last updated 3 days ago)_

HackerRank for Work lets you lock user access to prevent users from accessing the platform. You can lock individual users or multiple users.

# Prerequisite

You have **Team Admin** or **Company Admin** access.

# Locking user access

To lock user access:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Select your profile icon in the upper-right corner.

3.  Select **Teams Management** from the drop-down menu.

Locked users cannot log in and see an error message if they attempt to access the platform. 

**Note:** If a user belongs to only one team, [removing the user](https://support.hackerrank.com/articles/2203617737-manage-team-members#removing-a-team-member) from that team or [deleting the team](https://support.hackerrank.com/articles/2181136239-deleting-a-team) locks the user. 

# Unlocking user access

You can unlock a user account to restore access to the platform at any time.

To unlock user access:

1.  Go to the **Users** tab.

2.  Locate the user you want to unlock.

3.  Click the more options menu under **Actions**.

4.  Select **Unlock User.**
