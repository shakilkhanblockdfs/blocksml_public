# Modify Entitlements for Developers Prerequisite Modifying entitlements for developers Available developer entitlements

_Last updated: Jan 28, 2026, 11:00 AM (Last updated 3 months ago)_

You can modify developer entitlements to control access to tests, interviews, questions, and candidates.

# Prerequisite

You must have created a team.

# Modifying entitlements for developers

To modify entitlements for developers in a team:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Select your profile icon in the upper-right corner.

3.  Select **Teams Management** from the drop-down menu.

# Available developer entitlements

You can manage the following entitlement categories for developers:

- Tests

- Interviews

- Questions

- Interview templates

- Interview scorecards

- Candidates

- Exports

## Test entitlements

- View Tests: If toggled on, developers can:

  - Edit Tests

    - Create Tests

    - Clone Tests

    - Delete Tests

  - View Test Insights

  - View Test Settings

    - View Test Invites Settings
