# Browser Recommendations

_Last updated: Nov 13, 2025, 2:11 PM (Last updated 5 months ago)_

The table below lists the supported browsers and operating systems to access the HackerRank for Work platform.

|                 |                                 |
|-----------------|---------------------------------|
| **Browser**     | **Supported Operating Systems** |
| Google Chrome   | Windows, macOS                  |
| Microsoft Edge  | Windows, macOS                  |
| Mozilla Firefox | Windows, macOS                  |
| Safari          | macOS                           |

For the best performance, use the latest version of [Microsoft Edge](https://www.microsoft.com/en-us/edge?ep=313&form=MA13M0&es=40), [Mozilla Firefox](https://www.mozilla.org/en-US/firefox/new/), [Google Chrome](https://www.google.com/chrome/?brand=CHBD&gclid=EAIaIQobChMI0NfL0Pna3wIVwQ0rCh26QAfcEAAYASAAEgIg2fD_BwE&gclsrc=aw.ds), or [Safari](https://www.apple.com/in/safari/). Using an outdated browser may affect platform performance or prevent some features from loading correctly.

For browser compatibility details specific to HackerRank Test and Interview, see:

- [Browser Recommendations for HackerRank Tests](https://candidatesupport.hackerrank.com/articles/3310749702-browser-recommendations-for-hackerrank-tests)

- [Browser Recommendations for HackerRank Interviews](https://candidatesupport.hackerrank.com/articles/8455316478-browser-recommendations-for-hackerrank-interviews)
