# Teamtailor - HackerRank Integration

_Last updated: Dec 28, 2024, 12:26 AM (Last updated 1 year ago)_

## Overview 

HackerRank integrates with Teamtailor for Tests to facilitate an easy and seamless hiring experience. As a part of the workflow, users can send HackerRank Test invitations from Teamtailor. This results in an efficient recruiting process for the hiring team and the candidate.

Refer to the following articles to set up various features of the Teamtailor - HackerRank Integration:

- [📄 Teamtailor - HackerRank Integration Configuration Guide](/articles/3927772988)
  \
- [📄 Teamtailor - HackerRank Integration: User Guide](/articles/6599779249)
  \
