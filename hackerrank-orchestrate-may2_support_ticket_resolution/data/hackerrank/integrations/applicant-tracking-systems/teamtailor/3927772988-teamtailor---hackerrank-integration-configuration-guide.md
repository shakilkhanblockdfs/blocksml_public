# Teamtailor - HackerRank Integration Configuration Guide

_Last updated: Dec 28, 2024, 3:42 AM (Last updated 1 year ago)_

To set up the Teamtailor integration, you have to reach out to their support team and ask them to set it up. The below steps elaborate on how you can do that.  

## Inside the HackerRank Platform

1.  Log in to HackerRank for Work with the Company Admin user account.  

2.  On the home page, click the drop-down next to the user icon in the top right corner.

3.  Click **Settings**.
