# Lever V2 - HackerRank Integration Test User Guide Prerequisites Inviting candidates to a HackerRank test Viewing test results in Lever

_Last updated: Mar 12, 2026, 3:31 PM (Last updated 1 month ago)_

This guide outlines how to use the Lever V2 - HackerRank integration to send HackerRank test invitations and review test results directly in Lever V2.

# Prerequisites

Before you begin, ensure you meet the following requirements:

- You have admin access to your HackerRank for Work account.

- You have admin access to your Lever account.

- The HackerRank integration with Lever V2 is complete. For more information, see [📄 Lever V2 - HackerRank Integration Guide](/articles/5647291926).

# Viewing test results in Lever

After a candidate completes a HackerRank test, Lever displays the results on the candidate profile.

To view the test results in Lever:

1.  Log in to your Lever V2 account using your credentials.

2.  Go to the **Candidates** tab.

3.  Select the candidate whose test feedback you want to view.

4.  Scroll to the **Feedback summary** section on the **Overview** tab to view the test feedback and download the PDF report.

    **Note:** You can also review the test feedback from the **Feedback** tab.
