# Integration Logs

_Last updated: Dec 13, 2024, 5:02 PM (Last updated 1 year ago)_

## Overview

Integration Logs in HackerRank for Work allows you to access the records of activity between HackerRank and your primary system of record. These logs provide insight into the integration's transactions. They are available in the Integrations tab of the Logs page providing transparency. Users can inspect and debug the logs with ease.

**Note:** HackerRank displays logs only for the API calls that are made to the endpoints listed in the [HackerRank API documentation](https://www.hackerrank.com/work/apidocs).

## Accessing Integration Logs

1.  Login into your HackerRank account with a **Company Admin** user role.

2.  Click on your profile icon located at the top-right corner of the HackerRank homepage, then select **Settings**.

3.  On the left pane, click on **Integrations** under the Company section. This will direct you to the Integrations settings page.

4.  From the view of active integrations, choose the specific integration for which you wish to view logs. For example Ashby from the below screenshot.

5.  After selecting an integration, you will see tabs such as Configuration, Token Activity, and Logs. Click on the **Logs** tab.

6.  In the logs section, you can view the integration activity logs. You can search logs by recruiter email or apply filters based on date, status, and request method to narrow down the logs you need to view.

7.  If needed, you can download the logs for offline analysis or record-keeping by clicking on the download icon located near the filter settings.You will be able to get log results for a custom date range for an ATS instance and download a report in CSV format.
