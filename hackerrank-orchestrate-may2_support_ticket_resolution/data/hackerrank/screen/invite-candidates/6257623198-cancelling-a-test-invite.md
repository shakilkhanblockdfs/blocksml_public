# Cancel a Test Invite

_Last updated: May 12, 2025, 4:37 PM (Last updated 12 months ago)_

HackerRank for Work allows you to cancel a test invitation that you have already sent to a candidate.

## Prerequisites

- You must have created the test.

- You must have already sent the invitation to the candidate.

## Canceling a test invitation

To cancel a test invitation:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test from which you want to cancel a candidate's invitation.

4.  Go to the **Candidates** tab.

The candidate(s) will be removed from the test’s invitation list. The number of canceled invitations is credited back to your company’s account.

## Candidate experience

Canceled candidates can no longer access the test. If they attempt to start the test, an error message is displayed.
