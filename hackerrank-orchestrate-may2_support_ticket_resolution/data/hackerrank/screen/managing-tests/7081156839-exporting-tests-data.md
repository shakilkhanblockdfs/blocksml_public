# Export Tests Data Prerequisites Exporting tests data

_Last updated: Sep 3, 2025, 11:42 AM (Last updated 8 months ago)_

You can export test-level data as a CSV file that includes test details, test settings, attempt statistics, and more.

# Prerequisites

You must have existing test(s) in your HackerRank account.

# Exporting tests data

To export tests data: 

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab. 

3.  (Optional) Refine the list by applying filters in the left pane. You can filter by Labels, Owner, Role, Work Experience, Created At, and Certified Assessments.
