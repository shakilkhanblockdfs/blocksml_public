# Revoke Test Access

_Last updated: Jun 26, 2025, 12:59 PM (Last updated 10 months ago)_

HackerRank for Work allows admins to revoke test access from specific users or teams.

## Prerequisites

- You must be the test owner or have **Editor** access.

- The test must already be shared with the users or teams using the **Share** option.

## Revoking test access

To revoke access to a test for a specific user or team:

1.  Log in to your HackerRank for Work account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test from which you want to revoke access.

4.  Click **Share.**
