# Lock a Test Prerequisites Locking a test Unlocking a test

_Last updated: Feb 19, 2026, 2:46 PM (Last updated 2 months ago)_

Locking a test prevents changes to its questions and settings. Only test owners and company administrators can lock or unlock a test.

When a test is locked, you cannot:

- Edit test settings

- Change the order of questions

- Add new questions

- Modify existing questions

**Note:** You cannot delete a locked test, but you can archive it.

# Prerequisites

You must have an existing test in your HackerRank account.

# Locking a test

To lock a test:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test you want to lock.

4.  Click the more options menu (⋮) in the top right corner

5.  Select **Lock**.

# Unlocking a test

**Note:** To make changes to a locked test, you must first unlock it.

To unlock a test:

1.  Open the test you want to unlock.

2.  Click the more options menu (⋮) in the top right corner.

3.  Select **Unlock**.
