# Download Tests as PDF Prerequisites Downloading test as PDF

_Last updated: Sep 1, 2025, 12:08 PM (Last updated 8 months ago)_

You can download a HackerRank test as a PDF to share or review offline. 

# Prerequisites

You must have an existing test in your HackerRank account.

# Downloading test as PDF

To download the test as a PDF: 

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test you want to download.

4.  Click the more options menu (⋮) in the top right corner.

5.  Select **Download as PDF**.
