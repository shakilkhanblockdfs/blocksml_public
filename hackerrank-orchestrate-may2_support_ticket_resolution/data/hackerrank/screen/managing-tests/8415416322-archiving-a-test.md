# Archive a Test Prerequisites Archiving a test Restoring an archived test

_Last updated: Sep 25, 2025, 12:19 PM (Last updated 7 months ago)_

Archiving a test helps you manage your test library by moving unused tests out of the **Active Tests** tab. An archived test is no longer available for inviting candidates or publishing. You can restore an archived test at any time.

# Prerequisites

You must have an existing test in your HackerRank account.

# Archiving a test

To archive a test:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab. 

3.  Open the test you want to archive.

4.  Click the more options menu (⋮) in the top right corner.

5.  Select **Archive**.\

# Restoring an archived test

To restore an archived test: 

1.  Open the test you want to restore.

2.  Click the more options menu (⋮) in the top right corner.

3.  Select **Restore**.

Once restored:

- The test moves back to your list of Active Tests.

- Candidate invites are re-enabled.

**Note: **

- You can also restore the test by clicking the **Restore Test** button at the top of the test page.
