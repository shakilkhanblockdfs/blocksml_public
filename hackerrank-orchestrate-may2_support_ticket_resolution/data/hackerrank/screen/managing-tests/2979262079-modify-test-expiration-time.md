# Modify Test Expiration Time Prerequisite Modifying test expiration time

_Last updated: Mar 10, 2026, 9:05 PM (Last updated 2 months ago)_

You can define a start and end time (including time zone) to control when a test remains active.

Candidates can attempt the test only during the active period. After the test expires:

- Invited candidates cannot access the test.

- The **Invite** button is disabled.

- You cannot invite new candidates.

# Prerequisite

You must have created a test.

# Modifying test expiration time

To modify the test expiration time:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test you want to update.

4.  Go to **Settings \> General**.
