# Create a Test Creating a Test Additional test actions

_Last updated: Dec 3, 2025, 6:52 PM (Last updated 5 months ago)_

You can create a test in HackerRank to assess candidates for specific skills, experience levels, or job roles. Depending on your selection, HackerRank recommends relevant questions or allows you to build the test from scratch.

# Creating a Test

To create a test:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Click **Create Test**.

## Creating a test with job description

You can generate a skills-based assessment by using your job description. HackerRank analyzes the job description to identify the required skills and experience level, then recommends relevant questions from the HackerRank library. This helps you save time and ensures that the test aligns with the role requirements.

![createatestwithJD.gif](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1757090547211-createatestwithJD.gif?Expires=253370764800&Signature=ev~PYhvto1pBBK6awMMGuuJa5FBlLt34eftiaXihZ77FiMCt1kjC1k2uC~JVn8zFBBZTyN11ja1qrQf9yosShpkTrm3SdWFZrwzeUcg~ExFN7LYda-GRtpq0uEaQBgQonuJDr4HyLyCgTS3HFS5KDNZWzIBpPbn3GiGlE2p36tU-Hmo6fgcd-wkXGVWxzIJYVz~~mbL9S0zIYSr1ym~x~n-TiSIKLVG4muMGCcWts-gceXvXNj1MntmkBQpe1Ix6jpf49HrAHnTFnX9q6cutyA6Cz8zirf9IWID7T~T6mufNxNA3dZGkObQw6UW8elmZDrha5ZLzxe~JjoV0BYIAFw__&Key-Pair-Id=K3NV4LZ47N8M46)

To create a test with a job description:

1.  Select **Create Test with Job Description**.

2.  Copy and paste the job description into the input field.

3.  Click **Submit**.

4.  Review the auto-generated skills and questions in the **Test Preview** panel.

5.  Click **Create Test** to continue.

6.  (Optional) Edit the test:

    - To add sections and questions, see Manage Sections in Tests.

    - To delete a question, select the **Delete** icon next to the question.

    - To rename the test, click the edit icon next to the test title.

    - To change the test duration, click the edit icon next to the test duration.

7.  Click **Publish** to make the test available to candidates.

## Creating a test based on a specific role

You can create a test using predefined roles in HackerRank. Each predefined role includes mapped skills and curated questions from the HackerRank Library.

To create a test based on a specific role:

1.  Select a predefined role from the available options. You can search or scroll to find it.

2.  Review the auto-generated skills and questions in the **Test Preview** panel.

3.  Click **Create Test** to continue.

4.  (Optional) Edit the test:

    - To add sections and questions, see Manage Sections in Tests.

    - To delete a question, select the **Delete** icon next to the question.

    - To rename the test, click the edit icon next to the test title.

    - To change the test duration, click the edit icon next to the test duration.

5.  Click **Publish** to make the test available to candidates.

## Creating a custom test

Custom tests let you design assessments that meet specific hiring needs. You can create a test from scratch to evaluate candidates for particular roles, skill levels, or experience. Use this option when you need a highly tailored assessment.

For example, if you want to hire a Quality Assurance Engineer with five years of experience and expertise in creating optimal test automation scripts, you can create a custom test with questions that assess the required skills and experience.

You build the test by adding your own questions or by selecting questions from the HackerRank Library or your Company Library.

![CreateCustomTest.gif](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1757090627479-CreateCustomTest.gif?Expires=253370764800&Signature=cV3RxyAztCE~AmGuvKB-HxmsgcA5xYuPaQYSLpfG-f683qozlIq3CYZzwx0~evpddtEZyO7D1Q7KA7JLgv39kD3FDBDT0fmQnEUpdJSIsvijKInYMc17Ddnhk1OP82NT98bqpid7YKSgqe2~CFY9w21dbk3VLejbfmmS3tXI7lkVh6MwmrZtTAko7L6f~SBO1hK1otgEqUW78cbUKg5RU0hf-XllTRSTac27I~zAtTzrz5Guz5uEEbbUNfiGnjgSH~Xsny1pl~t2GgUfogHyPK1lrJHu4PZUjLagdGxKJRULmo10bMFxqq0yvlzEbvzO-JV84u09251O6DAuAZnNpQ__&Key-Pair-Id=K3NV4LZ47N8M46)

To create a custom test:

1.  Scroll to the bottom of the role list and select **Other**.

2.  Click **Create Test**.

3.  Configure the test:

    - To add sections and questions, see Manage Sections in Tests.

    - To delete a question, select the **Delete** icon next to the question.

    - To rename the test, click the edit icon next to the test title.

    - To change the test duration, click the edit icon next to the test duration.

4.  Click **Publish** to make the test available to candidates.

## Preview a test

You can preview a test from the candidate’s perspective by selecting **Try Test**. This action opens the complete test experience in a new window.

![image.png](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1757672035758-image.png?Expires=253370764800&Signature=JO8WOl8aDu27~bYrc1M-f6uHHdH~HoiRx6dBbimlbCA3JJYIeXsT9a48znQw-5LjVjoGMwyYDsU6-DbWSYUuG1ortcnFXwf7b65uWA2ncvmXWE4TlUQescvdhVQT5pZh6pQkYoqB7I-BZd6fHH4H7rKVZwpZoCbbXhlsCte3atfQwoeTX0QwfuHc26HZzKbAWN3LUVt~6GML7jYIY1AKq13-koSdViMWtEgQnBLmCwWjNdHHAcF2W4Toow6wzYdooIhTbIujURfUnf6TS1AoREFPuEpOXkJS0ehLsuk-ZIw3tqGB6LlNre25dBsYhT7rY62wRrx-fedHJrJ8JlEssw__&Key-Pair-Id=K3NV4LZ47N8M46)

You can also copy the test URL and share it with team members who have a HackerRank for Work license so they can open the link and preview the test themselves.

If a team member does not have a HackerRank for Work license, they cannot access **Try Test**. However, you can still benchmark the test or allow them to experience it by inviting them as a candidate. For more information, see [📄 Invite Candidates to a Test](/articles/6027855406).

HackerRank applies the following attempt rules when you preview or invite a candidate:

- **Using Try Test with the prefilled email:** When you select **Try Test** and keep the prefilled email unchanged, the preview does not consume an attempt.

- **Using Try Test with changes to the prefilled email:** When you select **Try Test** and replace the prefilled email with another email, the preview consumes an attempt.

- **Inviting an external candidate**: Inviting an external candidate consumes an attempt when the candidate launches the test.

- **Inviting an internal employee**: If you invite an internal employee who does not have a HackerRank for Work license, the invitation consumes an attempt when the employee launches the test.
