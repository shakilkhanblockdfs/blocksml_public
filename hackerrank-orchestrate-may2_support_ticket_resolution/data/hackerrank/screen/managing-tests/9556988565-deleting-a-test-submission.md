# Delete a Test Submission Prerequisites Deleting a test submission

_Last updated: Sep 3, 2025, 11:32 AM (Last updated 8 months ago)_

If you want a candidate to reattempt a test, you can delete the candidate’s test submission. This action permanently removes the candidate’s previous attempt, including all proctoring records, and resets their test status to **Invited**.

**Note:** It is recommended that you do not delete a candidate test submitted if the test is in progress.

# Prerequisites

You must have a test submitted by at least one candidate in your HackerRank for Work account.

# Deleting a test submission

To delete a test submission: 

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab. 

3.  Open the test that contains the submission you want to delete.

4.  Go to the **Candidates** tab.
