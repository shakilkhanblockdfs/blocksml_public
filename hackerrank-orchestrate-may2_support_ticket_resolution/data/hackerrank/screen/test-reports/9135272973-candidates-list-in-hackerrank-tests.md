# View Candidates List for a Test Viewing the candidates list

_Last updated: Oct 14, 2025, 7:34 PM (Last updated 6 months ago)_

The candidates list in the test interface provides a centralized view of all invited candidates and their test statuses. It helps you track candidate progress, review reports, and manage invitations in one place.

# Viewing the candidates list

To view the list of invited candidates for a specific test:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test from which you want to view or manage candidate invitations.

4.  Go to the **Candidates** tab to view the list of all candidates who have been invited or have participated in the test.
