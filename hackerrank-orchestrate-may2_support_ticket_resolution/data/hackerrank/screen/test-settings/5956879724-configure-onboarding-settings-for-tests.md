# Configure Onboarding Settings for Tests Prerequisites Configuring onboarding settings

_Last updated: Apr 22, 2026, 10:20 AM (Last updated 4 days ago)_

HackerRank allows you to configure the onboarding settings for a test to collect candidate details and display test instructions before the test begins. Most settings are optional and customizable to fit your hiring needs.

# Prerequisites

You must have already created a test.

# Configuring onboarding settings

To configure onboarding settings:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Go to the **Tests** tab.

3.  Open the test you want to configure.

4.  Go to **Settings \> Onboarding.**

    In the **Onboarding** section, you can:

    - [Configure Candidate Details](https://support.hackerrank.com/articles/5956879724-configure-onboarding-settings-for-tests#configure-candidate-details-6)

    - [Enable Disclaimer](https://support.hackerrank.com/articles/5956879724-configure-onboarding-settings-for-tests#enable-disclaimer-11)

5.  Click **Save** **Changes** to apply your changes.

## Configure Candidate Details

You can choose the candidate information fields you want to collect before the test begins. This helps you capture relevant data such as full name, work experience, gender, and more. By default, only essential fields are enabled.

To configure candidate details:

![(15FPS) Updates to Candidate Input List and Test Settings.gif](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1759745738339-%252815FPS%2529UpdatestoCandidateInputListandTestSettings.gif?Expires=253370764800&Signature=Qxzva48vx5t3~uokTXnFep~G8dkxnCiLKxpL1UKz1RuGCPvUofolYEfQHcsQ58FerQXnOYqjRebc-S049IszYHtdBBhklpAYk8BBrZToRtnKFDrS9A3xZqkCaGL8ZCJlgWctpjwcuczie9OyKvkdflaGCkcS9qlfowwUHNaNvKDcJF30QM8t0y1~mmVdhuARIo2bz5jTjOY6P1BZzpjU5hDtiwukzzHzzDhndYmcVfb7Aw-7g2n-EQM1-8ytfoCTgzcILn4yT0uNtHj25uCXo~6OB9wjTEqChmu4F3ElqW7jJ5IVmwy8iqQRzdg7--vOmssy9lUFtLtBmGqPpRJl5w__&Key-Pair-Id=K3NV4LZ47N8M46)

1.  In the **Candidate Details** section, select the default fields you want to collect.

2.  Click **Add a custom field** to include custom fields.

    - In the **Add Custom Field** dialog box:

      - Enter the **Field name**.

      - Select a **Type** from the dropdown list. Available types include Text Field, Radio Button, Checkbox, Dropdown, and File.

      - If you select Radio Button, Checkbox, or Dropdown, define the options. Click **Add Option** to include more choices.

      - Select **Mark as optional** if the field is not required.

      - Click **Add** to save the field.

    **Note:** If the field you are adding is predefined in your company’s central test settings, it appears in the field drop-down as you type the field name. Select the existing field to avoid duplication. You cannot change the field type for a predefined field.

    ![onboarding add fields.gif](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1769349726214-onboardingaddfields.gif?Expires=253370764800&Signature=OJcd~Sxvr4PJlFYtxvqdDWtBhQEMIa8Xuk1aW26WQDwGxDrxQkBgM309wkreE9yq8SOoJKR9qb8GbaLbV4xSeRjmZ4ZxnNBd6kv74tDvBe7ilenmqf3B-VgoHuARjrqjDGemVR12ysfvvgJ0rn2BdKWXFkkPzg~fVVF9CZGmn1QPMtahiueUPfNP~BMIfxWdXrU-5wY-Gkf8pS4IeSm4U3tFIUM5IzUTe7dHvc3AfqApEnQy6dssAbM63PnWqRKPgqHXz5~zrfXMeEodoix9Wb4kuHA1ZqgwhpexU5hEe~iBuPDUth7HRG5fwpPSdj6aqCF0NOJU6ogZxqJMl9o6hQ__&Key-Pair-Id=K3NV4LZ47N8M46)

3.  Drag and drop fields to change the order in which they appear to candidates.

4.  Click **Try Test** to see how the onboarding form appears to candidates.

## Enable Disclaimer

You can enable a disclaimer to ensure candidates acknowledge test integrity requirements before starting a test. When enabled, a declaration with a checkbox appears on the test login page. Candidates must select the checkbox to proceed.

The editor supports rich text formatting such as bold, italic, underline, and hyperlinks.

To enable the disclaimer:

![image.png](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1775458827993-image.png?Expires=253370764800&Signature=rd53Wm3BcYWrP~xYdD7uNnAfCwaVEUuwlCwDZFHkZAEJlu-R5p1xHm54J~Xryvv0EV35pVYDdBAFmMkXnoF1M3h~Jmx02K785wKslA3rF3RUMTZ3MsaeevxJIlw-fzfrr5UDBNS1CewFGx~4JAHlbnhJt0v06Dcd5lp5LHcGegQjVEUwRcuK9OrZ3O2VVxt5H4Zft2Sk4A1n8Rzi6BNtoxKOxr6cfGAbE0LXrFfp4s1XiPhGMx5JT9L5OW4a1M7Ah4xe2DBURRf9hvZqPk2kXx6jipWyAsEG8aQAHGCNI3S5raS9PjglxO9v-1gbtdIl0cJrZb1Uf1h5Ps0lbSc2IA__&Key-Pair-Id=K3NV4LZ47N8M46)

1.  Turn on the **Disclaimer** toggle.

2.  Edit the disclaimer text in the text box.

## Configure Test Instructions

You can add custom instructions to guide candidates before they begin the test. This helps set expectations and ensures a smooth test experience. 

You can either add test-specific instructions or use global templates. Include details such as test expectations, platform tips, and best practices.
