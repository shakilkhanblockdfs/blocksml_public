# How to Allow Candidates to Retake a Test?

_Last updated: Apr 9, 2025, 11:23 AM (Last updated 1 year ago)_

For a candidate to be able to retake your test (which is submitting another attempt after the most recent one), you can either [📄 Reinviting Candidates to a Test](/articles/1002936098)to the same test (when you do so, a candidate's test status initializes to "Invited").

**Note**: When Candidates try to access a Test for a reattempt, sometimes, a browser-level cache may disallow them from logging in to the test. To overcome this issue, it is recommended that your candidates access the Test link using a new browser window.

To re-invite candidates to a test, you can select one or more candidates from the **Candidates** tab in your test and click on **Invite Selected** as shown below. For more information on re-inviting candidates, refer to the [📄 Reinviting Candidates to a Test](/articles/1002936098).

**Note**: When you re-invite a candidate, the most recent attempt by the candidate is archived. A candidate can be re-invited as many times as needed.
