# HackerRank SkillUp Licenses

_Last updated: Dec 13, 2024, 5:06 AM (Last updated 1 year ago)_

## SkillUp 

SkillUp is a comprehensive platform designed to help organizations manage and develop their technical talent effectively. It offers a range of features and tools to support developers and employers in their growth, workforce planning, and talent mobility efforts.

## Licenses For SkillUp

HackerRank SkillUp offers the following user license types:

- Platform License

- Org License 

- Individual License

Each license type provides access to specified features and functionality within SkillUp. All licenses must be assigned to individual named users within your organization. 

## Platform License

The Platform License provides access to all the SkillUp features and functionality included with both an Org License and an Individual License. Access to functionality is granted through SkillUp Org and SkillUp Individual licenses.

## Org License 

The Org License provides users with access to the following features:

- View company overview

- View company talent directory

- Export Individual User details from the talent directory

- Assign goals to Individual Users

- View the Skill Profile of all individual users

- Add users 

- Export the list of individual users in user management

- View and store certification assessments for use in connection with SkillUp

- View raw reports on certification assessments

Org Licenses are intended for use by Admin, Managers, and Executives. Admin users are granted access to a corresponding Recruiter license in the HackerRank platform to facilitate their use of SkillUp. 

## Individual License

The Individual License provides users with access to the following features:

- Explore the role directory and skills directory

- Execute learning plans

- Take skill badging assessments and role certification assessments

- Download Role Certificates

- Manage their Skill Profile

Individual Licenses are generally intended for use by individual developers within your organization who will access the skills development features of SkillUp.
