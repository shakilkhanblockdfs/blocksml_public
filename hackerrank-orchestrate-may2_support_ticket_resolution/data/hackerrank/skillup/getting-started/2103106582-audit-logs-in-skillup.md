# Audit Logs in SkillUp How to request audit logs

_Last updated: Jan 28, 2026, 10:13 AM (Last updated 3 months ago)_

Audit Logs provide a detailed record of organizational activity in SkillUp. Organizations can use audit logs to support governance, security monitoring, and compliance requirements.

Audit logs capture key events performed within SkillUp, including:

- User additions, removals, and role changes

- Badge, certification, and assignment-related actions

- Administrative configuration changes

The information included in an audit log varies based on the request and the selected time period.

# How to request audit logs

To request audit logs, contact the SkillUp Support team at [skillup-support@hackerrank.com](mailto:skillup-support@hackerrank.com).

**Note:** Only administrators can request audit logs for a SkillUp account. This restriction ensures that only authorized users can access organization-wide data.

## Sample template

You can use the following template to request audit logs:

***Subject:*** *Request for SkillUp audit logs*

\

*Hello SkillUp Support Team,*

*I am requesting Audit Logs for our organization’s SkillUp account to support internal audit and compliance requirements.*

*Request details:*

- *Organization name: \[Your organization name\]*

- *Requested time period: \[start date – end date\]*

- *Relevant event types (if applicable): \[Categories of events to include. For example, user authentication, administrative actions, or assignments\]*

*Thank you for your assistance.*

*Kind regards,\
\[Your full name\]\
\[Your role/title\]\
\[Your organization\]\
\[Work email address\]*

**Note:** If you have questions about audit logs or are unsure what information to provide, contact [skillup-support@hackerrank.com](mailto:skillup-support@hackerrank.com).
