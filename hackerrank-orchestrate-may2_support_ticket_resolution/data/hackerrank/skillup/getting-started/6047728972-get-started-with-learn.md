# Get Started with Learn

_Last updated: Apr 23, 2025, 11:40 AM (Last updated 1 year ago)_

The Learn experience in SkillUp offers a structured, flexible, and hands-on learning environment. It helps developers improve their technical skills through guided practice and AI-powered support.

 It includes the following features:

- Self-paced, contextual learning with guided examples and interactive challenges.

- Real-time AI tutor to support coding practice and concept reinforcement.

- Skill badges that validate proficiency and progress.

## Intended audience

The Learn experience is designed for: 

- Individual developers seeking to upskill in specific technologies.

- Organizations aiming to promote talent growth by helping employees gain and demonstrate new skills.

## Using Learn

To begin using the Learn experience:

1.  Log in to **SkillUp** using your credentials.

2.  Go to the **Learn** tab.

3.  Select a skill (for example, Python, Angular).

4.  Select a concept from the **Foundational**, **Intermediate**, or **Expert** track.

5.  Read the lesson material with examples, explanations, and guidance from the AI tutor.

6.  Use the built-in AI tutor to ask questions or get help as needed.

7.  Start the associated coding challenge when you are ready. The AI tutor remains available for support.

8.  Submit your completed solution.

9.  Complete all challenges in a track to earn a Skill badge.

## Common use cases

- A developer preparing for a new backend role wants to refresh their Python fundamentals.

- A backend engineer transitioning to a full stack position uses the React learning path to build frontend skills.

- A manager encourages the employee to upskill and earn a badge to demonstrate readiness for an internal opportunity.

## Best practices

To get the most out of the Learn experience:

- Use the AI tutor to deepen your understanding instead of skipping directly to the challenge.

- Set aside short, regular time blocks to complete modules at your own pace.

## Known limitations

The following limitations apply to the current experience:

- The AI tutor is available only during Learn and Badge experiences. It is not accessible during timed certification assessments.

- Badges are skill-specific and currently available only for certain learning paths.

## FAQs

**Do I need to complete a badge before taking a certification?**

No. Completing a badge is helpful but not a prerequisite for certification

**Are challenges timed?**

No. Learn and badge-related challenges are untimed. Only certification assessments are timed.

**Can I retake a module if I make a mistake?**

Yes. You can reattempt modules as needed and take the time required to master the content.

**Where can I find my badge after I earn it?**

Badges will appear in your **SkillUp profile** upon completion of the Learn track.
