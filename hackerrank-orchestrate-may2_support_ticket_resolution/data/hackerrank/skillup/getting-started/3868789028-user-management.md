_Last updated: Feb 23, 2026, 12:23 PM (Last updated 2 months ago)_

The **User Management** section provides access to all users within your organization. You can view, add, and manage user accounts based on assigned roles and permissions.

# Accessing User Management

To access user management:

1.  Log in to your **HackerRank for Work** account using your credentials.

2.  Click your profile icon in the upper-right corner and select **Admin Panel**.

# Adding a new user

To add a new user:

1.  Click **Add New Users**.

    ![image.png](https://assets.usepylon.com/e6a58e21-be80-4777-9eaf-f73beeee94d9%2F1760452180280-image.png?Expires=253370764800&Signature=HbMJ7X3rATY4ssHToXjUV4iM9g9S0dHC~UNA2qRUqtokfcYlv-JLALH3tv6cWtHh84hbPBLY3d4HJ8iBRENcsn9YlITXRqawcILXIvpRKv-g~QpjEkwKrNQhWTAeMReLiFdyzm~jcUAB0NdOyPznBslU8k8uOIdkHUV3x3pz6rzgM0ts1LZnRaPmJV-gj1Es5qsWVs08gbuo9ndUdizMmGriHh~3syYJxgYjPz-FoLC~K36CwklPe8OWg8dMV37UB1VyqzQAAkvjcrDkoNDyttEc7h7g1G4fTOLU0iu6pONuBlbLEaCURVDdS3kzk3i52-ttl4fVDOrhuFIIHKjXiw__&Key-Pair-Id=K3NV4LZ47N8M46)

2.  Choose an option:

    - **Invite user**: 

      1.  Select a role:

          - Individual License:

            - **Individua**l: Explore skills and roles, earn badges and certifications, and create a skill profile. 

          - Organization License:

            - **Manager**: Access all individual features, assign tasks to team members, view insights, and export reports.

            - **Admin**: Access company-wide insights, reports, and user management tools.

      2.  Click **Next**.

      3.  Update the following fields:

          - First Name

          - Last Name(Optional)

          - Email

          - Job Title

          - Manager

      4.  Click **Next**.

      5.  Select certification:

          - **Allow users to select:** Let users choose their first certification.

          - **Assign a specific certification**: Choose a certification from the drop-down menu.

      6.  Click **Add User**.

    - **Upload CSV**: Upload a CSV file to add users in bulk.
